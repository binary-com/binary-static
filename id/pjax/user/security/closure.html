Menonaktifkan Akun

Sebelum menonaktifkan akun, Anda harus melakukan hal berikut:

1. Tutup semua posisi berjalan

Jika Anda memiliki akun riil Binary, kunjungi Portfolio untuk menutup posisi berjalan.

Jika Anda memiliki akun riil DMT5, akses akun tersebut untuk menutup posisi berjalan.

2. Tarik saldo Anda

Jika Anda memiliki akun riil Binary, kunjungi Kasir untuk menarik saldo Anda

Jika Anda memiliki akun riil DMT5, kunjungi Dashboard DMT5 untuk menarik dana Anda.
