Layanan DNS publik Cloudfare memberi Anda penjelajahan web yang lebih cepat dan aman.

Apa itu DNS

Domain Name System (DNS) memungkinkan Anda untuk mengakses suatu situs melalui alamat web yang mudah digunakan seperti www.binary.com, daripada alamat IP yang rumit.

Mengapa Anda harus menggunakan 1.1.1.1 daripada layanan DNS default ISP Anda

Layanan DNS default yang disediakan oleh ISP Anda bisa lambat dan tidak aman. Binary.com merekomendasikan layanan DNS 1.1.1.1 Cloudflare yang memberi Anda cara yang lebih cepat dan lebih pribadi untuk menjelajah internet.

Siapkan 1.1.1.1 pada perangkat Anda sekarang
