Qu'est ce qu'un DNS

Le Système de Nom de Domaine (DNS) vous permet d'accéder au site internet à travers une adresse internet facile à utiliser comme www.binary.com, au lieu d'une adresse IP compliquée.

Pourquoi vous devriez utiliser 1.1.1.1 au lieu de votre service DNS par défaut de votre ISP

Les services DNS standards fournis par votre ISP peuvent être lents et non sécurisés. Binary.com recommande d'utiliser le service DNS de Cloudflare 1.1.1.1 qui vous donne une façon de naviguer sur internet plus rapide et plus confidentielle.
